About Climate Saver

The Mission:

Climate Saver is a free to play web-based video game made to educate it's players about climate change what they can do to help the ongoing dillema.

The game features 4 different environments to play in:

  • The first environment is located at a beach where the player has to pick up people's trash before it reaches the ocean in order to save the sea animals from being effected by the trash.
  • The second environment is located in a city and the goal of this environment is...
  • The third environment is located at a forest. The goal of this level is to stop a fire from spreading before it reaches the nearby civilians' homes
  • The fourth environment is located at an icey place. The goal of this environment is to save the ice from melting by protecting it from the sun.

Education:

Throughout every level there will be facts and statistics provided about climate change. At the end of every level the player must take a quiz to test their knowledge about climate change gained from the game.

Technologies and Skills

The technologies used to create Climate Saver:

  • Phaser.io for game development.
  • Tiled for environment creation.
  • Github for version control and team collaboration.
  • Jira for project management.

The skills needed to create Climate Saver:

  • Javascript in order to code the game in Phaser.io's framework.
  • Github in order to properly track changes in the code and to allow for collaboration amongst peers.
  • Graphic design in order to create environments and assets.
